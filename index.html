Tentang Saya

Lulusan SMK.

Perkenalkan, nama saya Athallah Abhi Pramana. Saya lahir di Bandung pada tanggal 19 Juli 2005. Saya memiliki minat besar dalam desain dan coding, serta memiliki pengalaman dalam memanfaatkan teknologi AI dan bekerja dalam bentuk Tim, selain itu saya juga memahami bahasa Ingris.

Selama 2 tahun terakhir ini Saya menekuni bidang web developer. Saya mempelajari berbagai teknologi dan bahasa pemrograman seperti HTML, CSS, JavaScript, Python, Java, dan PHP, serta pengelolaan basis data menggunakan MySQL dan MongoDB. kemudian saya juga mempelajari beberapa framework seperti bootstrap, TailwindCSS dan Flask, serta tools desain seperti Figma, dan juga version control Menggunakan GIT melalui platform GitHub.
